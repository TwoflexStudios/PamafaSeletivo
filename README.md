- [@Weslley Araujo](https://www.linkedin.com/in/weslley-araujo-b38271228/)


## Rodando os testes

Para rodar o projeto, rode os seguintes comandos dentro da pasta do projeto

```bash
  npm install
```
```bash
  npm run-script dev
```
ou

```bash
  yarn install
```
```bash
  yarn dev
```
